# Cozy English Window

A calm technology application for tracking your 30-day English study challenge. Watch a beautiful, peaceful image slowly reveal itself as you complete each day of study.

## 🎯 Features

- **30-Day Challenge**: Track your daily English study progress through a frosted glass window
- **Beautiful Imagery**: Each day reveals a new portion of a cozy, peaceful scene from Unsplash
- **Calm Design**: Warm color palette and gentle animations designed to reduce stress
- **Persistent Progress**: Your progress is automatically saved in your browser
- **One Study Per Day**: Prevents multiple clicks on the same day
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **No Dependencies**: Single HTML file with embedded CSS and JavaScript

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Click "Study Done" each day after completing your English study session
3. Watch as the frosted glass window gradually clears to reveal a beautiful image
4. After 30 days, start a new window with a fresh image

## 🎨 Design Philosophy

This application follows **Calm Technology** principles:
- Peaceful, warm color palette (Off-white, Latte Beige, Soft Brown)
- Smooth, soothing animations (1.5-second transitions)
- Minimal, uncluttered interface
- Gentle encouragement instead of aggressive gamification
- Sustainable daily habits without pressure

## 💾 Data Storage

Your progress is saved locally in your browser using `localStorage`:
- Current day number
- Revealed grid cells
- Background image URL
- Last study date (to prevent multiple clicks per day)

## 🌐 Browser Compatibility

Works on all modern browsers:
- Chrome/Chromium
- Firefox
- Safari
- Edge
